Create REPO

cd /F  : goes to F drive
ls : lists everything

cd into folder

to start a repo create a folder with a file

git init - initilizes git repository within local folder

git add .  : adds all files for git to track

git commit -am 'attach message here' : this command commits your changes with a message

go to git website to create repo

git remote add origin 'address to repo' : tells local folder address to online repo as the origin

git push origin master : pushes all changes to online repo


Create Branch

git checkout -b 'branch name' : creates branch and moves to that branch

git checkout 'branch name' : moves to existing branch

git push origin 'branch name'  : pushes all changes to online repo



Merging Branches

git merge 'branch name' : merges changes from another branch into existing branch

Check Status of Folder

git status
